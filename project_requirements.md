Требования к проекту

Admin:
• Авторизуется под своими кредами, передавая токен авторизации
• Может создавать пользователей с разными ролями (ROLE_USER, ROLE_CREDIT_SECRET)
• Может отправлять /get запрос на получение всех пользователей
• Может удалять пользователей по id

User:
• Авторизуется под своими кредами (которые создали на админе)
• Может создать два банковских счета
• Может пополнять баланс
• Может переводить деньги между своими счетами
• Может переводить деньги другому пользователю на его банковский счет
• Может получать историю своих транзакций
• Может запросить кредит, если создан с ролью "ROLE_CREDIT_SECRET"

Role:

User:
• Может логиниться под своими кредами
• Создавать два банковских счета
• Пополнять банковские счета
• Переводить деньги между своими счетами
• Переводить деньги на банковский счет другому пользователю
Credit Secret:
• Обладает теми же правами что и User
• Существует возможность запрашивать кредит на один из банковских счетов
• Если кредит запрошен на один банковский счет и одобрен, запрашивать кредит на второй
счет невозможно

1. Authorization:
• Можем логиниться под валдиным кредами всех созданных юзеров
• Админ может авторизоваться под кредами:
"username": "admin", "password": 123456

2. Admin:
username:
• Только латиница
• Минимум 3 символа
• Максимум 15
• Разрешены цифры и буквы
• Спец. символы запрещены
password:
• Только латиница
• От 8-ми символов
• Минимум 1 заглавная буква
• Минимум 1 маленькая буква
• Минимум 1 спец. символ
• Минимум 1 цифра

При удалении юзера, вся информация о не удаляется, в том числе банковские счета, истории
транзакций и кредиты

3. Account
• Максимум возможно создать 2 банковских счета
• Сумма пополнения (deposit) - минимально 1000, максимально - 9000
• Сумма перевода - минимально 500, максимум - 10000
• Перевод возможен между своими счетами
• Перевод возможен на счет другого пользователя
• Пополнение не ограничено по количеству
• Переводы не ограничены по количеству, ограничены только по деньгам

4. Credit
• Кредит имеет право взять только юзер с ролью "Credit"
• Минимальная сумма кредита - 5000, максимальная 15000
• Кредит можно погасить только один раз, единовременно и на всю сумму
• Пополнение по одноразовым суммам - отсутствуют
• Кредит возможно взять только на один из счетов
• Если кредит взят на другом счету или не погашен - новый кредит взять невозможно
Типы операций:
1. deposit (Пополнение)
2. transfer_in (Куда)
3. transfer_out (Перевод из)
